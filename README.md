# Sistema de Consulta de Funcionários - Excel

## Sobre o Projeto

Projeto desenvolvido em Microsoft Excel para consulta de funcionários através do ID.

## Funcionalidades

* Consulta por ID
* Busca automática utilizando PROCV
* Tratamento de erros com SEERRO
* Consulta de Nome, Cargo, Departamento, Salário, Data de Admissão, E-mail e Celular
* Indicadores de RH

## Indicadores

* Total de Funcionários
* Maior Salário
* Menor Salário
* Média Salarial

## Ferramentas Utilizadas

* Microsoft Excel
* PROCV
* SEERRO
* CONT.VALORES
* MÁXIMO
* MÍNIMO
* MÉDIA

## Objetivo

Praticar análise de dados, consultas automatizadas e construção de sistemas simples utilizando Excel.
